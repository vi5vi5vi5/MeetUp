# models/

Структуры данных проекта — по образцу MedFlow (`models/*.h`): простые
`struct` без поведения, `id = -1` означает «ещё не сохранён» (репозиторий
присвоит id при `save`).

| Модель | Что описывает |
|---|---|
| `User` | Аккаунт: логин, отображаемое имя, производные пароля (PBKDF2-хеш + соль + число итераций — сам пароль не хранится), версия аватарки. `publicJson()` — представление для клиента без секретов. |
| `Session` | Сессия входа: токен (он же ключ, живёт в HttpOnly-куке), пользователь, срок действия. |

Планируются: `PersonalRoom` (личная конференция: код, название, пароль,
владелец) — вместе со следующей итерацией.

Хранением занимаются репозитории из [`../interface`](../interface/README.md),
бизнес-правилами — [`../services`](../services/README.md).
