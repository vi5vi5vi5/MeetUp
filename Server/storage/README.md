# storage/

Слой хранения данных. Пока папка пустая — заготовка под SQLite
(модуль Qt `QtSql`, драйвер `QSQLITE`).

## Как планируется устроить

- `Database.*` — открытие файла БД, создание схемы при первом запуске,
  миграции при изменении схемы.
- Репозитории — по одному на сущность (`UserRepository`,
  `ConferenceRepository`, `MessageRepository`, …). **Весь SQL живёт здесь**;
  сервисы из [`../services`](../services/README.md) зовут методы репозиториев
  и про SQL не знают.

Такое разделение позволит подменять хранилище (например, in-memory для
тестов) не трогая бизнес-логику — тот же приём, что в MedFlow
(`interface/Abstract` → `InMemory` / `sqlite`).
