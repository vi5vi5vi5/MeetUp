# web/

Статичный веб-клиент для тестирования сервера — одна страница `index.html`
(getUserMedia → canvas → WebSocket). Отдаётся встроенным `HttpFileServer`
на **http://localhost:8081**, отдельный веб-сервер не нужен.

Что умеет: вход в комнату по коду, видео участников, список участников,
общий чат с временными метками.

Дизайн временный и будет заменён. Сюда же позже добавятся страницы
регистрации и настройки профиля — `HttpFileServer` отдаёт любой файл из
этой папки по его имени (`/login.html` → `web/login.html`).
